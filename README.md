# 🍽️ Food Delivery MongoDB Project

## 📌 Overview
This project demonstrates MongoDB queries for a food delivery system including:
- Customers
- Restaurants
- Orders
- Payments
- Reviews
- Delivery Agents

## 🛠️ Technologies Used
- MongoDB
- MongoDB Shell / Compass

## 📂 Project Structure
- data/ → sample datasets
- queries/ → all MongoDB queries

## 🚀 Features
- 100+ MongoDB queries
- Filter queries
- Aggregation pipelines
- Real-world food delivery scenario

## 📊 Collections Used
1. Customers
2. Restaurants
3. Menu
4. Orders
5. Payments
6. Delivery Agents
7. Reviews

## ▶️ How to Run
1. Import JSON files into MongoDB
2. Open MongoDB Shell / Compass
3. Run queries from `queries/` folder
